## 📱 Responsive Behavior

### Desktop (1280px+)
- Hero: Side-by-side layout
- Skills: 3-column grid
- Experience: Full timeline
- Projects: 3-column grid

### Tablet (768px - 1279px)
- Hero: Stacked with centered content
- Skills: 2-column grid
- Experience: Compact timeline
- Projects: 2-column grid

### Mobile (< 768px)
- Hero: Single column, smaller image
- Skills: 1-column grid
- Experience: Simplified cards
- Projects: 1-column grid
- Navigation: Hamburger menu

## ✨ Animation Highlights

1. **Page Load**: Sections fade in sequentially
2. **Scroll**: Elements slide up when visible
3. **Hover**: Cards lift with shadow
4. **Click**: Ripple effect on buttons
5. **Typing**: Cursor blinks, text types
6. **Particles**: Floating background animation
7. **Theme**: Smooth color transitions
8. **Navigation**: Smooth scroll to sections
